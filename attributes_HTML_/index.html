Atributos

El atributo href

La etiqueta a define un hipervinculo

El atributo href especifica la URL de la pagina.

Visit w3scools

Atributo src

La etiquet img se utiliza para incrustar imagnes en una página html

el atributo src especifica la ruta de la imagen que se mostrara.

Existe dos formas de especificar la url de src

1: URL absoluta, enlaces a una imagen externa

Ejemplo: src:https://w3schools.com

2: URL relativa, enlaces a una imagen alojada en elsitio web

Ejemplo: src:=/imagenes/ima_girls.jpg
